Microinteraction Enhancement: Beta

BETA - INFORMATION

Triggers

    Microinteraction 1 and 2

  • Hover over, off and click on the ‘Online store: ikea.us’ button

    • Microinteraction 3

  • Hover over and off the 'Go shopping' button

Rules

Microinteraction 1 and 2

1. Hover over

  • The cursor changes into a pointer
  • When the mouse hovers over the 'Online store: ikea.us' button, its background fades into a light yellow.

2. When clicked

  • The 'Find online store' search tool pulls up when clicking the 'Online store: Ikea.us' button.

3. Hover off

  • When hovering off, the button's background changes back to yellow.

Microinteraction 3

1. Hover over

  • The cursor changes into a pointer
  • When the mouse hovers over the 'Go shopping' button, the arrow icon next to the text floats to the right

2. Hover off

  • When hovering off, the arrow icon next to the 'Go shopping' button floats to the left, back to the original position
