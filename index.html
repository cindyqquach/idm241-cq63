Microinteraction Enhancement: Alpha

ALPHA - INFORMATION

Triggers

  • Hover over a tab on the menu bar

Rules

  • The cursor changes into a pointer
  • A blue underline is drawn from left to right twice. The first time it moves fast, doubling the speed of the second time.
  • On the second time the line connects, it stops, creating an underline underneath the menu tab name.
  • The interval between the time each line is drawn is 0.5 seconds.
